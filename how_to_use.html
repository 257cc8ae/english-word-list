このアプリの使い方

📥 1. 単語リストを登録する

「新しい単語帳を登録」 から、ChatGPTで生成したJSON形式の単語リストを貼り付けて保存します。
JSONは word, meaning, example_en, example_ja, partOfSpeech を含む必要があります。

🤖 ChatGPT用プロンプト

次の英文に含まれる重要単語について、以下の形式のJSONを出力してください。日本語訳は自然なものにしてください。

【英文】
<ここにユーザーの英文を貼り付ける>
【出力形式】
[
    {
        "word": "単語",
        "meaning": "日本語の意味",
        "example_en": "元の英文からその単語が使われている例文",
        "example_ja": "その英文の自然な日本語訳",
        "partOfSpeech": "品詞（例：noun, verb, adjective など）"
    }
]

【条件】
- 難しめの語彙や学術的な語彙を優先的に取り上げてください
- JSON形式の構文エラーがないようにしてください（カンマや括弧の抜けに注意）
- 出力はJSONだけにしてください（説明や挨拶は不要）

📚 2. 単語帳を選んで学習する

ホーム画面に表示された単語帳の一覧から、学習したいリストを選びます。

🗣️ 3. 単語と例文を確認・発音練習する

単語と例文には読み上げボタンがあります。
「答えを見る」で意味や例文を確認し、「次へ」で進みます。
発音や例文のリズムをまねて覚えると効果的です。

💡 補足

  • 単語帳名にはスペースや記号は使えません（URLで使うため）
  • ローカルストレージに保存されるため、データはブラウザ内に保持されます
  • スマホでもPCでも同じように使えます
